# Rencana Penelitian: Evaluasi Usability Super-App Shopee Menggunakan SUS

## 1. Ringkasan

| Item | Keterangan |
|---|---|
| Judul | Evaluasi Dampak Usability Integrasi Layanan Pesan-Antar Makanan pada Super-App E-Commerce: Studi Kasus Perbandingan Shopee Reguler dan Shopee Food |
| Target Publikasi | Sinta 2 (Jurnal RESTI/Telematika) atau Scopus Q3-Q4 |
| Stack | Google Form, Python, Pandas, SciPy |
| Masalah | Integrasi layanan majemuk pada antarmuka aplikasi (*feature bloat*) berpotensi menciptakan beban kognitif berlebih (*cognitive overload*) bagi pengguna, merusak navigasi. |
| Solusi | Evaluasi kuantitatif komparatif (Shopee Reguler vs Shopee Food) menggunakan *System Usability Scale* (SUS) dengan desain uji *Within-Subjects*. |
