# Tahap 5: Penyusunan Manuskrip Jurnal Ilmiah

## Tujuan
Menyusun seluruh narasi latar belakang, landasan teori, eksekusi metolodogi, hingga ekstraksi komputasi statistik ke dalam sebuah draf naskah yang memenuhi standar gaya selingkung publikasi akademik IEEE (Sinta 2 / Scopus).

## Struktur Pengerjaan
Untuk memudahkan revisi silang, naskah tidak langsung digabung, melainkan dikerjakan secara modular di direktori `07-manuskrip/`:
- `00-outline.md`
- `01-abstrak.md`
- `02-pendahuluan.md`
- `03-tinjauan-pustaka.md` (Mengutip jurnal mutakhir terkait *super-app*)
- `04-metodologi.md`
- `05-hasil-analisis.md`
- `06-kesimpulan.md`
- `07-daftar-pustaka.md` (Format *IEEE Style* dengan jurnal-jurnal Indonesia asli dari Google Scholar)

## Hasil / Output
Sebuah *file* gabungan bernama `naskah-jurnal.md` yang siap di-*copy-paste* langsung ke *template Microsoft Word* milik penyelenggara publikasi tanpa perlu memikirkan ulang struktur isinya.
